# Fast Food Ordering Kiosk

A fast food ordering kiosk system built with C# and WinForms, inspired by real-world self-service kiosk systems used in fast food restaurants.

## Features

- Tabbed menu categories — Meals, Drinks, Desserts
- Add to cart functionality with real-time order summary
- Remove items and clear cart via context menu
- Checkout with order total calculation
- Custom styled UI with professional color scheme
- About page with system information

## Technologies Used

- Language: C#
- Framework: .NET WinForms
- IDE: Visual Studio

## How To Run

1. Clone the repository
2. Open the `.sln` file in Visual Studio
3. Build and run the project
4. Navigate through menu tabs and add items to cart
5. Click Checkout to complete the order
